## Added GitHub Issue Templates

### Summary

Introduced GitHub Issue Forms to standardize issue creation on GitHub.

### Changes

- Added `.github/ISSUE_TEMPLATE/feature.yml`
- Added `.github/ISSUE_TEMPLATE/config.yml`
- Enabled `blank_issues_enabled: true`
- Standardized the GitHub Issue format using Issue Forms

### Reason

The project has standardized its GitHub Issue structure.
GitHub Issue Forms were introduced to make issue creation more consistent while maintaining compatibility with the existing AI-assisted workflow.

Blank issues remain enabled during the initial evaluation period to preserve flexibility while the workflow is being validated.

### Result

- Human-created issues now follow the same structure as AI-generated issues.
- GitHub Issue Forms are available for feature and task requests.
- Blank issues can still be created during the evaluation period.
