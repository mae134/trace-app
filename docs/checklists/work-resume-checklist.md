# 作業再開チェックリスト

このチェックリストは、人間がAIコーディングエージェントとの共同開発で作業再開時の確認漏れを防ぐために使用します。

必要に応じて、AIコーディングエージェントにもこのチェックリストを参照させてください。

## ワークフロー

現在の状況確認
↓
GitHub Issue 確認
↓
関連ドキュメント確認
↓
現在のブランチ確認
↓
作業再開

## 使用タイミング

作業を再開する前に確認してください。

## 作業再開前

- [ ] GitHub Issue を確認した
- [ ] Approved Design を確認した（存在する場合）
- [ ] 現在のブランチを確認した
- [ ] 未コミットの変更がないことを確認した
- [ ] 最新のコミット履歴を確認した
- [ ] 関連する Pull Request を確認した（存在する場合）
- [ ] 関連ドキュメント（README / Playbook / ADR など）を確認した（必要な場合）
- [ ] 次に実施するタスクを確認した
