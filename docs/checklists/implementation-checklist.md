# 実装チェックリスト

このチェックリストは、人間がAIコーディングエージェントとの共同開発で実装工程の確認漏れを防ぐために使用します。

必要に応じて、AIコーディングエージェントにもこのチェックリストを参照させてください。

## ワークフロー

GitHub Issue
↓
Approved Design
↓
実装（AIコーディングエージェントまたは人間）
↓
動作確認
↓
コミット

## 使用タイミング

実装開始前および実装完了後に確認してください。

## 実装開始前

- [ ] GitHub Issue を確認した
- [ ] Approved Design を確認した（存在する場合）
- [ ] 実装範囲を確認した
- [ ] Out of Scope を確認した
- [ ] 現在のブランチを確認した
- [ ] 未コミットの変更がないことを確認した

## 実装完了後

- [ ] 必要な確認（例: lint / test / build / 動作確認）を実施した
- [ ] GitHub Issue の内容どおりに実装したことを確認した
- [ ] Out of Scope の変更が含まれていないことを確認した
- [ ] AI実装時は実装レポートを確認した
- [ ] コミット前に変更内容を確認した
